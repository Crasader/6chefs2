# encryptor

## 必要環境
 - Mac OS
 - Atom
 - script (Atomのパッケージ)

## usage
1. encryptor.cppをAtomで開く
2. Cmd+i (Atomのscriptの実行)

## 注意
git管理下のファイルが書き変わるので、コミットしないように！
`Resouces/config/Encrypted.json` というファイルの存在が暗号化されている翔子として、ゲーム内のプログラムで使われる
